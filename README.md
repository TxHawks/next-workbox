# next-workbox

> Next.js plugins for workbox and PWA

# Installation

```sh
npm install --save next-workbox
```
or
```
yarn add next-workbox
```

# Usage

```js
// next.config.js
const withWorkbox = require('next-workbox')
module.exports = withWorkbox()
```
